# 🌍 Oil Price Impact Analysis Dashboard

## 📊 Project Overview

This project analyzes crude oil prices over time and evaluates the impact of major global events, particularly the **Russia-Ukraine war**, on oil price trends.

An interactive dashboard has been built using Streamlit, allowing users to explore the data dynamically through filters and visualizations.

## 📌 Key Insights

* Oil prices increased significantly after the war, rising from an average of approximately **47 USD** before the war to about **83 USD** after the war.
* This represents an increase of roughly **36 USD per barrel**, indicating a strong impact of geopolitical events.
* Monthly trends show sharp spikes around early 2022, aligning with the war timeline.
* Yearly averages confirm that recent years experienced higher price levels and increased volatility compared to earlier periods.

## 📊 Analysis Summary

This project highlights how external geopolitical events influence global oil markets. The dashboard enables users to explore:

* Short-term fluctuations (daily & monthly trends)
* Long-term patterns (yearly averages)
* Event-based impact (before vs after war)

This analysis demonstrates that geopolitical events can have a measurable and significant effect on oil prices. The project showcases how data-driven insights can be used to understand real-world economic changes.
